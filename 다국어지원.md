## 언어 설정 방식
- 설정 화면에서 수동 선택 가능
- `auto` 모드: 기기 시스템 언어를 자동 감지하여 적용
- 미지원 언어일 경우 영어(en)로 폴백

## 다국어 적용 범위

### UI 전체
- 설정 화면 (메뉴, 옵션, 설명)
- 스플래시 화면 (초기화 메시지, 태그라인)
- WayBack 버튼 상태 (복귀, 안내 중, 도착, 복귀 중, 완료)
- 내비게이션 패널 (경로 안내, 거리, 시간)
- 다운로드 관련 메시지 (진행률, 완료, 오류)
- 안전 경고 및 알림

### 장소 정보 바텀시트
- 장소 유형 배지 (restaurant→음식점, cafe→カフェ 등 28종)
- OSM 상세 라벨 (영업시간, 전화, 웹사이트 등 8개 항목)
- 로딩 메시지
- 번역 불가 안내 메시지

### AI 기능
- Gemini AI 응답 언어 지시 (각 언어별 지시문)
- 장소 설명 2단계 번역 (로컬 언어 → 대상 언어)
- 장소 이름/주소 번역
- 경로 분석 및 안전 조언
- 교차로 방향 안내

### 지도 번역
- ML Kit 오프라인 번역 (우선)
- Gemini AI 온라인 번역 (폴백)
- 번역 캐시 (TranslationCacheService)

## 관련 파일

### WayBack 앱
| 파일 | 역할 |
|------|------|
| `lib/utils/app_localizations.dart` | 전체 번역 키-값 관리 (13개 언어 × 200+ 키) |
| `lib/providers/app_state_provider.dart` | 언어 설정 저장/로드 |
| `lib/services/gemini_service.dart` | AI 응답 언어 지시, 번역 메서드 |
| `lib/services/ml_kit_translation_service.dart` | ML Kit 오프라인 번역 |
| `lib/services/translation_cache_service.dart` | 번역 결과 캐시 |

### 홈페이지 (Home_Page)
| 파일 | 역할 |
|------|------|
| `client/src/i18n.js` | i18next 설정, 언어 메타데이터 (14개 언어) |
| `client/src/locales/{언어코드}/translation.json` | 각 언어별 번역 파일 (676줄) |
