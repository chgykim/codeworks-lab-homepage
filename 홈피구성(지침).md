# CodeWorks Lab [Safe Way] — v6.0 (Way 14 — WayVision + WayMoments + WayEnvelope 추가)

## Safe Way 철학

```
"산에서 길을 잃었다. 진짜 무서웠다.
 이런 앱이 있었으면... 하고 만든 게 WayBack이다.
 그리고 삶의 다양한 순간에서 '이 앱이 있었으면!' 하는 것들을 만든다."
```

### 이 기준을 충족하는 앱만 만든다:
1. **실제 상황에서 "이 앱이 있었으면!" 하는 순간이 존재**
2. **대기업이 안 하거나 못 한다**
3. **1인이 만들 수 있다**
4. **사람들이 돈을 낸다는 증거가 있다**

### 법적 방패
**"우리는 판단하지 않습니다. 기록하고, 보여주고, 도울 뿐입니다."**

### 3. WayVision — AI 시각·건강 보조 [개발 예정] ★3순위

> **"글씨가 안 보인다. 이 약 뭔지 모르겠다. 읽어줬으면 좋겠다."**

**법적**: "시각 보조 도구 + 건강 정보 참고" — 의료기기 아님

| | |
|---|---|
| DNA 순간 | 노안으로 글씨 안 보임, 약 성분 모름, 약물 중복 위험 |
| 없으면? | 약물 중복 복용 → 부작용/사망, 유통기한 못 읽음 → 식중독 |
| 핵심 연령층 | **40-80세 노안 세대 (글로벌 수십억 잠재 유저)** |
| 벤치마크 | **Magnifier 앱 (무료), Be My Eyes (무료), Seeing AI (무료)** |
| 대기업 위협 | 무료 확대경 존재하지만 **AI 성분분석+약물 상호작용+음성 읽기 조합 없음** |
| 스모킹건 | **카메라 확대 + AI OCR + 약품 분석 + 약물 상호작용 경고 + 음성 읽기 = 세계 최초 조합** |
| 구독 전환 포인트 | 노안은 매일 겪음. 약 먹을 때마다 사용 = 구독 필수 |
| 충성 고객 확충 | 약국/이비인후과, 시니어 커뮤니티, 자녀→부모 선물, 아시아 고령화 시장 |
| 수익 모델 | **구독형: 월 3,900원 / 연 29,000원** (무료: 카메라 확대 + 기본 OCR) |
| 출시 | **최우선 개발** |

#### 핵심 기능
- 카메라 실시간 확대 (2x~10x) + 자동 초점/밝기/대비
- AI OCR 텍스트 인식 (흐린 글씨/악필 보정)
- 성분표/영양성분 자동 분석 + 건강 점수(0~100)
- 유통기한 자동 인식
- 제품 전면 패키지 인식 → 자동 제품 DB 검색
- **조제약/시판약 성분 분석** (약품명, 용량, 부작용, 금기)
- **약물 중복·상호작용 경고** (킬러 기능)
- **질환 기반 위험 분석** (고혈압/당뇨/심장/신장 환자 맞춤 경고)
- 초대형 글씨 재표시 (최대 60pt)
- 음성 읽기 기능 (TTS)

#### 3순위 선정 근거 (최대 직접 수익)

| 기준 | WayVision | 다른 앱 대비 |
|------|-----------|-------------|
| **시장성** | 글로벌 40~80대 노안 시장 (수십억) | 압도적 1위 |
| **구독전환률** | 건강/안전 긴급성 + 매일 사용 | 최상위 |
| **글로벌 확장성** | 미국(다약제), 일본(초고령화), 중국(약품시장), 유럽(건강관심) | 전 세계 통용 |

#### Firebase Functions 엔드포인트
- `/ocrText` — Gemini Vision OCR
- `/detectProductFront` — 전면 패키지 인식
- `/searchProductDB` — 제품 DB 검색 (식약처, FDA, PMDA, OpenFDA)
- `/analyzeIngredients` — 성분 위험 분석 + 건강 점수
- `/analyzeDrug` — 약품 성분/부작용 분석
- `/drugInteractionCheck` — 약물 중복·상호작용 경고

**수익 잠재력**: ★★★★★ | **개발**: 8-10주 | **출시**: 3순위

### 2. WayMoments — AI 감성 자동 기록 [개발 예정] ★2순위 (바이럴 마케팅 엔진)

> **"좋은 모임이었는데, 뭘 이야기했는지 기억이 안 난다. 저절로 기록됐으면..."**

| | |
|---|---|

| 없으면? | 추억이 사라짐, 중요한 대화 내용 소멸 |
| 핵심 연령층 | **10-60세 (전 연령) — 친구 모임, 데이트, 가족 식사, 동창회, 회식, 동호회** |
| 벤치마크 | **Day One: $35/yr, Journey: $30/yr** |
| 경쟁 | 일기 앱은 많지만 **자동 기록 + AI 감성 분석 + 에세이 생성 = 없음** |

| 수익 모델 | **구독형: 월 3,900원 / 연 29,000원** (2주 무료 체험) |
| 출시 | **WayBack 다음 2순위** |

#### 핵심 기능
- 모임/세션 자동 감지 및 기록 시작
- 대화 내용 AI 요약
- 사진·장소·참석자 자동 태그
- 감정 자동 분석 (표정, 대화 톤, 키워드)
- 세션 종료 시 **AI 감성 타임라인 생성**
- 세션 종료 시 **AI 감성 에세이 자동 생성**
- 기념일/추억 리마인더
- **설정 화면 하단: 사용설명서 + 개발자 홈페이지(https://rustic-sage.web.app/) 링크**

#### 2순위 선정 근거 — 바이럴 마케팅 플랫폼 분석

##### 1. 소셜 바이럴 효과 (CAC ≈ $0)

```
[모임 장면]
A가 식당에서 WayMoments 켜놓음
→ B, C, D, E가 "이거 뭐야?" 질문
→ 모임 끝나고 AI 에세이 공유
→ "나도 깔아볼래" = 무료 다운로드
→ 2주 무료 체험 → 자기 모임에서도 사용
→ 그 모임의 F, G, H, I가 또 "이거 뭐야?"
= 바이럴 계수 > 1 (지수적 성장)
```

| 지표 | 수치 | 근거 |
|------|------|------|
| 평균 모임 인원 | 4-6명 | 식당/카페/술자리 |
| 월 평균 모임 횟수 | 4-8회 | 10-60세 사회 활동 |
| 모임당 노출 인원 | 3-5명 | 본인 제외 |
| 노출→다운로드 전환 | 20-30% | "직접 봤으니까" (일반 광고 2% 대비 10배) |
| **1명이 월간 유입시키는 신규 유저** | **3-8명** | 4회 모임 × 5명 노출 × 25% = 5명/월 |
| **바이럴 계수** | **1.5-3.0** | 1명이 1.5-3명 유입 = **지수적 성장** |

> **비교**: 일반 앱 바이럴 계수 0.3-0.5, **WayMoments 1.5-3.0 (3-10배)**
> **핵심**: 유료 마케팅 비용 $0으로 지수적 성장 가능한 유일한 앱

##### 4. 2주 무료 체험 → 손실 회피 전환

```
Day 1: 첫 모임 기록 → "신기하네"

> 심리학적으로 **손실 회피 > 공포** (Kahneman & Tversky, 1979)

## 스모킹건 분석 — 전체 포트폴리오

### 핵심 전략: WayMoments (바이럴 엔진) + WayVision (수익 엔진) + WayPulse (건강 엔진) + WaySound (접근성 엔진)

**4대 엔진 체계 — 글로벌 14개 언어 출시 기준:**

```
🔥 WayMoments (2순위) = 바이럴 마케팅 엔진 (CAC $0, 유저 유입)
   → 모든 앱에 유저 공급 (설정 → 홈페이지 → Way 14 발견)
   → 바이럴 계수 1.5-3.0 (지수적 성장)

💰 WayVision (3순위) = 최대 직접 수익 엔진
   → 글로벌 18억+ 노안 시장, 구독 전환율 최상

❤️ WayPulse (4순위) = 건강/생존 엔진
   → Android 완성, 건강앱 전환율 9.4%

🔊 WaySound (8순위) = 접근성/대형 시장 엔진
   → 15억 난청 인구, 시장 규모 최대
```

**왜 WayMoments가 4대 엔진에 포함되나:**

1. **WayMoments = 유일한 바이럴 엔진** — 바이럴 계수 1.5-3.0 (다른 앱은 0.3-0.5)
2. **WayMoments = Way 14 유입 퍼널** — 설정 화면 → 홈페이지 → 13개 앱 발견
3. **WayMoments = CAC $0 성장** — 모임 자리에서 자연 홍보, 유료 마케팅 불필요
4. **WayMoments = 최광범위 연령** — 10-60세 (WayVision 40-80, WayPulse 45-65)
5. **WayVision = 글로벌 최대 TAM** — 40-80세 노안 인구 **18억+**, 모든 국가에서 통용
6. **WayVision = 세계 최초 조합** — 카메라 확대 + AI OCR + 약물 상호작용 + 음성 읽기 = 경쟁 앱 없음
7. **건강 앱 = 가장 높은 구독 전환** — 건강앱 다운로드→구독 전환율 **9.4%** (전 카테고리 1위)
8. **WayPulse 이미 완성됨** — 다른 앱은 0에서 시작, WayPulse는 **Android 출시만 하면 됨**
9. **14개 언어 = 글로벌 접근성** — 영어+중국어+스페인어+힌디어로 세계 인구 60%+ 커버
10. **mHealth 시장 폭발 성장** — $37.5B (2024) → $86.4B (2030), **CAGR 14.8%**

> **WayMoments가 유저를 데려오고, WayVision + WayPulse가 수익을 만든다**
> **= 마케팅 비용 $0으로 건강앱 수익 극대화 전략**

```

  └→ TTS 음성 읽기 → WaySound (자막 음성), WayER (의료 회화 음성)

  ├→ Health Connect 연동 → (독립, 향후 WayRest 수면 데이터 연동 가능)
  ├→ 긴급 SMS 시스템 → WayPanic, WayBack SOS

  ├→ AI 감성 분석 → WayStory (자서전 감성 보강)

#### 왜 WayMoments가 2순위인가 (수정된 분석)

**이전 12순위 판단의 오류:**
- ❌ "DNA 적합성 약함" → ✅ 추억 소멸은 **돌이킬 수 없는 상실** (비가역적)
- ❌ "구독 전환율 낮음" → ✅ 2주 감성 데이터 축적 → **손실 회피** 전환율 높음
- ❌ "일기 앱 경쟁 치열" → ✅ **자동 기록 + AI 에세이 생성** 조합은 경쟁 없음
- ❌ "직접 수익만 평가" → ✅ **간접 가치 (바이럴 + 크로스셀)가 직접 수익보다 크다**

**2순위 선정 핵심 근거:**
1. **바이럴 계수 1.5-3.0** — 유일하게 지수적 유저 성장 가능한 앱 (CAC $0)
2. **10-60세 전 연령** — 가장 넓은 사용자 풀 (다른 앱은 특정 연령대에 한정)
3. **Way 14 유입 퍼널** — 설정 → 홈페이지 → 13개 앱 교차유입 생성
4. **2주 무료 체험 → 손실 회피** — 감성 데이터 인질 효과로 높은 구독 전환
5. **빠른 개발** — 8주 (Gemini + 카메라 + 위치 = 기존 기술 재활용)

> **WayMoments를 먼저 출시하면, WayVision 출시 시 이미 유저 풀이 형성되어 있다**
> **= 마케팅 비용 $0으로 WayVision 런칭 가능**
