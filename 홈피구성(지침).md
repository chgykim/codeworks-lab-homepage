아무리 좋은 의도라도 '진단(Diagnosis)', '처방(Prescription)', '치료(Cure)' 같은 뉘앙스를 풍기면 바로 제재 대상이 됩니다. 따라서 모든 앱의 정체성을 \*\*"생활 편의 도구(Utility)"\*\*와 \*\*"자기 관리 기록장(Journal)"\*\*으로 철저히 리포지셔닝(Re-positioning) 했습니다.



법적 리스크를 0%로 만든 CodeWorks Lab의 \[Safe Way 10] 라인업입니다.



🛡️ CodeWorks Lab \[Safe Way 10] 라인업

앱의 목표를 \*\*'의료'\*\*에서 \*\*'라이프스타일/기록'\*\*으로 변경했습니다.



1\. WayBack (웨이백) - 안전 귀가 \& 여행

변경 없음 (안전함)



목표: 오프라인 지도 및 길 찾기, 위치 기반 여행 가이드.



법적 포인트: 네비게이션은 의료기기가 아닙니다. (통과)



2\. WayFit (웨이핏) - 식단 기록 \& 칼로리 계산

목표: 음식 사진을 찍어 나만의 \*\*'식단 일기'\*\*를 쓰고, 대략적인 영양 정보를 얻음.



수정 사항: "혈당 관리/당뇨 식단" 언급 삭제 → \*\*"건강한 식습관 형성"\*\*으로 변경.



법적 포인트: 영양 정보를 알려주는 것은 '정보 제공'이지 의료 행위가 아닙니다.



3\. WayMuscle (웨이머슬) - 홈트레이닝 코치

목표: 집에서 정확한 자세로 운동하도록 돕는 'AI 운동 파트너'.



수정 사항: "재활/교정" 단어 삭제 → \*\*"바른 자세 운동"\*\*으로 변경.



법적 포인트: 헬스장 트레이너 역할을 AI가 하는 것은 웰니스(Wellness)입니다.



4\. WayBrain (웨이브레인) - 두뇌 유산소

목표: 퀴즈와 대화를 통해 뇌를 즐겁게 하는 '두뇌 트레이닝 게임'.



수정 사항: "치매 예방/진단" 언급 절대 금지 → \*\*"두뇌 활력 유지"\*\*로 변경.



법적 포인트: 닌텐도 두뇌 트레이닝 게임처럼 '오락/교육' 카테고리로 접근합니다.



5\. WayView (웨이뷰) - 스마트 돋보기

목표: 작은 글씨를 크게 보여주고 읽어주는 '시각 보조 도구'.



수정 사항: "시력 교정" 언급 삭제 → \*\*"독서 확대경"\*\*으로 정의.



법적 포인트: 안경(의료기기)이 아니라, 카메라 확대 기능(유틸리티)입니다.



6\. WaySound (웨이사운드) - 대화 자막 비서

목표: 잘 안 들리는 대화를 실시간 텍스트로 보여주는 '자막 생성기'.



수정 사항: "보청기" 단어 사용 금지 → \*\*"실시간 받아쓰기 메모장"\*\*으로 정의.



법적 포인트: 청력 보정이 아니라, 단순히 소리를 글자로 바꿔주는 '편의 기능'입니다.



7\. WayLog (웨이로그) - 약물 알림장 (구 WayCare)

이름 변경: Care(케어)는 의료 느낌이 나서 \*\*Log(기록)\*\*로 변경.



목표: 영양제나 약 먹는 시간을 잊지 않게 알려주는 '알람 시계'.



수정 사항: "약물 상호작용 경고" 삭제 → "복용 이력 기록" 기능에 집중.



법적 포인트: 단순히 시간을 알려주는 알람 앱은 규제 대상이 아닙니다.



8\. WaySpot (웨이스팟) - 바디 관찰 일기 (구 WaySkin)

이름 변경: Skin(피부과) 느낌을 빼고 \*\*Spot(점/부위)\*\*으로 변경.



목표: 몸의 변화(점, 상처 등)를 사진으로 찍어 '시간 순서대로 보여주는 앨범'.



수정 사항: "피부암 분석" 절대 불가 → \*\*"변화 비교 뷰어"\*\*로 변경. (판단은 사용자가 함)



법적 포인트: 사진을 정리해 주는 갤러리 앱은 의료기기가 아닙니다.



9\. WayRest (웨이레스트) - 수면 소리 녹음 (구 WaySleep)

이름 변경: Sleep(수면클리닉) 대신 \*\*Rest(휴식)\*\*로 부드럽게.



목표: 잠자는 동안의 소리(코골이, 잠꼬대)를 녹음해 주는 '수면 레코더'.



수정 사항: "무호흡증 진단" 삭제 → \*\*"밤사이 소리 확인"\*\*으로 변경.



법적 포인트: 녹음기 앱은 의료기기가 아닙니다.



10\. WayStory (웨이스토리) - 자서전 메이커

변경 없음 (안전함)



목표: 대화를 통해 인생 이야기를 책으로 정리해 줌.



법적 포인트: 출판/작문 보조 도구입니다.



🏛️ CodeWorks Lab의 법적 방패 (Safety Shield)

이 10개 앱을 관통하는 하나의 메시지는 이것입니다.



"우리는 판단하지 않습니다. 기록하고, 보여주고, 도울 뿐입니다." (We do not judge. We record, display, and assist.)



AI(제미나이)에게도 "이 사진이 암인가요?"라고 물어보면, \*\*"저는 의료 전문가가 아닙니다. 하지만 사진을 날짜별로 정리해 드릴 수는 있습니다."\*\*라고 대답하도록 만들면 됩니다.



사장님, 이렇게 이름을 바꾸고 목표를 \*\*'기록(Log)'\*\*과 \*\*'보조(Assist)'\*\*로 한정하니 마음이 훨씬 편하시죠? 이게 바로 \*\*롱런(Long-run)\*\*하는 비결입니다.
