# CodeWorks Lab [Safe Way] — v6.0 (Way 14 — WayVision + WayMoments + WayEnvelope 추가)

## WayBack DNA란?

```
"산에서 길을 잃었다. 진짜 무서웠다.
 이런 앱이 있었으면... 하고 만든 게 WayBack이다.
 모든 기능은 '있으면 좋은 것'이 아니라 '없으면 큰일나는 것'이다."
```

### 이 DNA를 가진 앱만 만든다:
1. **실제 상황에서 "이 앱이 있었으면!" 하는 순간이 존재**
2. **없으면 진짜 곤란하다** (일기장/기록장 = 탈락)
3. **대기업이 안 하거나 못 한다**
4. **1인이 만들 수 있다**
5. **사람들이 돈을 낸다는 증거가 있다**

### 법적 방패
**"우리는 판단하지 않습니다. 기록하고, 보여주고, 도울 뿐입니다."**

### 1. WayVision — AI 시각·건강 보조 [개발 예정] ★1순위

> **"글씨가 안 보인다. 이 약 뭔지 모르겠다. 읽어줬으면 좋겠다."**

**법적**: "시각 보조 도구 + 건강 정보 참고" — 의료기기 아님

| | |
|---|---|
| DNA 순간 | 노안으로 글씨 안 보임, 약 성분 모름, 약물 중복 위험 |
| 없으면? | 약물 중복 복용 → 부작용/사망, 유통기한 못 읽음 → 식중독 |
| 핵심 연령층 | **40-80세 노안 세대 (글로벌 수십억 잠재 유저)** |
| 벤치마크 | **Magnifier 앱 (무료), Be My Eyes (무료), Seeing AI (무료)** |
| 대기업 위협 | 무료 확대경 존재하지만 **AI 성분분석+약물 상호작용+음성 읽기 조합 없음** |
| 스모킹건 | **카메라 확대 + AI OCR + 약품 분석 + 약물 상호작용 경고 + 음성 읽기 = 세계 최초 조합** |
| 구독 전환 포인트 | 노안은 매일 겪음. 약 먹을 때마다 사용 = 구독 필수 |
| 충성 고객 확충 | 약국/이비인후과, 시니어 커뮤니티, 자녀→부모 선물, 아시아 고령화 시장 |
| 수익 모델 | **구독형: 월 3,900원 / 연 29,000원** (무료: 카메라 확대 + 기본 OCR) |
| 출시 | **최우선 개발** |

#### 핵심 기능
- 카메라 실시간 확대 (2x~10x) + 자동 초점/밝기/대비
- AI OCR 텍스트 인식 (흐린 글씨/악필 보정)
- 성분표/영양성분 자동 분석 + 건강 점수(0~100)
- 유통기한 자동 인식
- 제품 전면 패키지 인식 → 자동 제품 DB 검색
- **조제약/시판약 성분 분석** (약품명, 용량, 부작용, 금기)
- **약물 중복·상호작용 경고** (킬러 기능)
- **질환 기반 위험 분석** (고혈압/당뇨/심장/신장 환자 맞춤 경고)
- 초대형 글씨 재표시 (최대 60pt)
- 음성 읽기 기능 (TTS)

#### 1순위 선정 근거

| 기준 | WayVision | 다른 앱 대비 |
|------|-----------|-------------|
| **시장성** | 글로벌 40~80대 노안 시장 (수십억) | 압도적 1위 |
| **구독전환률** | 건강/안전 긴급성 + 매일 사용 | 최상위 |
| **글로벌 확장성** | 미국(다약제), 일본(초고령화), 중국(약품시장), 유럽(건강관심) | 전 세계 통용 |

#### Firebase Functions 엔드포인트
- `/ocrText` — Gemini Vision OCR
- `/detectProductFront` — 전면 패키지 인식
- `/searchProductDB` — 제품 DB 검색 (식약처, FDA, PMDA, OpenFDA)
- `/analyzeIngredients` — 성분 위험 분석 + 건강 점수
- `/analyzeDrug` — 약품 성분/부작용 분석
- `/drugInteractionCheck` — 약물 중복·상호작용 경고

#### 왜 WayBack DNA인가
- WayBack: "길을 잃으면" → 안전 귀가 → **자연의 위험**
- WayVision: "글씨가 안 보이면" → AI가 읽어줌 → **시각의 위험**
- **40대 이상 누구나 겪는 노안 + 약물 안전 = 가장 넓은 시장**

**수익 잠재력**: ★★★★★ | **개발**: 8-10주 | **출시**: 최우선

### 2. WayPulse (웨이펄스) — AI 건강 코파일럿 [Android 완성]

> **"혼자 있는데 가슴이 두근거린다. 심박수가 140이다. 누가 좀 알아챠줬으면..."**

**법적**: "건강 기록 도우미" — 의료기기 아님 (삼성헬스/Apple Health 선례)

| | |
|---|---|
| DNA 순간 | 혼자 있을 때 심박수 폭주, 매일 혈압 추적, 건강검진 비교 |
| 없으면? | 위험 신호 놓침 → 돌연사, 수치 기록 없음 → 악화 |
| 벤치마크 | **Cardiogram: $99/yr, Welltory: $80-156/yr** |
| 대기업 위협 | 삼성헬스(무료)는 AI 코칭/긴급SMS/검진관리/암호화 없음 |
| 1인 개발 | **Android 100% 완성 (31세션)**, iOS 포팅만 필요 |

#### 현재 완성된 기능 (Android)
- **Health Connect 연동**: 삼성헬스/Google Fit → 심박수/걸음수/수면/혈압/산소포화도
- **실시간 생체 모니터링**: 30초 간격 + 배터리 최적화 자동 조절
- **위험 감지 → 긴급 SMS**: 연속 3회 위험 수치 → 가족에게 자동 GPS SMS
- **AI 건강 코칭**: Gemini 3 Flash (Firebase Cloud Functions) — 맞춤형 분석/리포트
- **건강검진 기록**: 국가검진 결과 연도별 비교 차트
- **가정 측정 기록**: 혈압/혈당/체중 트렌드 그래프 + 3일 연속 이상치 경고
- **AES-256 암호화**: SQLCipher + Android Keystore (100% 로컬 저장)
- **앱 잠금**: PIN/생체인증
- **6단계 온보딩**: 약관 → 인증 → 기본정보 → 질병 → 유전요인 → 목표

#### 출시를 위한 리프레이밍 (식약처/의료기기법 대응)

| 현재 표현 | 출시용 변경 |
|----------|-----------|
| "위험 감지 알고리즘" | "사용자 설정 범위 이탈 알림" |
| "서맥/빈맥/고혈압 위기" | "설정 범위보다 낮음/높음/크게 초과" |
| "AI 건강 분석" | "AI 생활 습관 리포트" |
| "의료 가이드라인 기반" | "일반 건강 정보 참고" |

#### 스모킹건 기능

**"AI 맞춤 코칭 + 긴급 SMS + 암호화 로컬 저장"** — 삼성헬스에 없는 3종 세트

### 최대 기회: WayVision + WayPulse + WaySound (3대 엔진)

**글로벌 14개 언어 출시 기준 — 왜 이 3개가 최대 수익원인가:**

1. **WayVision = 글로벌 최대 TAM** — 40-80세 노안 인구 **18억+**, 모든 국가에서 통용
2. **WayVision = 세계 최초 조합** — 카메라 확대 + AI OCR + 약물 상호작용 + 음성 읽기 = 경쟁 앱 없음
3. **건강 앱 = 가장 높은 구독 전환** — 건강앱 다운로드→구독 전환율 **9.4%** (전 카테고리 1위)
4. **건강 앱 = 가장 높은 수익** — Revenue Per Install $0.44 (전 카테고리 1위)
5. **WayPulse 이미 완성됨** — 다른 앱은 0에서 시작, WayPulse는 **Android 출시만 하면 됨**
6. **14개 언어 = 글로벌 접근성** — 영어+중국어+스페인어+힌디어로 세계 인구 60%+ 커버
7. **자녀→부모 선물 = 글로벌 보편** — 전 세계 어디서나 "부모님 건강"은 해지 불가
8. **mHealth 시장 폭발 성장** — $37.5B (2024) → $86.4B (2030), **CAGR 14.8%**
9. **핵심 3개 시장**: 미국(다약제+보험비), 일본(초고령화), 중국(14억 인구+스마트워치)

> **WayVision + WayPulse 2개 앱이 나머지 12개 합친 것보다 클 수 있다**
