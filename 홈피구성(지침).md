# CodeWorks Lab [Safe Way] — v5.0 (WayBack DNA + WayPulse)

## WayBack DNA란?

```
"산에서 길을 잃었다. 진짜 무서웠다.
 이런 앱이 있었으면... 하고 만든 게 WayBack이다.
 모든 기능은 '있으면 좋은 것'이 아니라 '없으면 큰일나는 것'이다."
```

### 이 DNA를 가진 앱만 만든다:
1. **실제 상황에서 "이 앱이 있었으면!" 하는 순간이 존재**
2. **없으면 진짜 곤란하다** (일기장/기록장 = 탈락)
3. **대기업이 안 하거나 못 한다**
4. **1인이 만들 수 있다**
5. **사람들이 돈을 낸다는 증거가 있다**

### 법적 방패
**"우리는 판단하지 않습니다. 기록하고, 보여주고, 도울 뿐입니다."**

### 2. WayPulse (웨이펄스) — AI 건강 코파일럿 [Android 완성]

> **"혼자 있는데 가슴이 두근거린다. 심박수가 140이다. 누가 좀 알아챠줬으면..."**

**법적**: "건강 기록 도우미" — 의료기기 아님 (삼성헬스/Apple Health 선례)

| | |
|---|---|
| DNA 순간 | 혼자 있을 때 심박수 폭주, 매일 혈압 추적, 건강검진 비교 |
| 없으면? | 위험 신호 놓침 → 돌연사, 수치 기록 없음 → 악화 |
| 벤치마크 | **Cardiogram: $99/yr, Welltory: $80-156/yr** |
| 대기업 위협 | 삼성헬스(무료)는 AI 코칭/긴급SMS/검진관리/암호화 없음 |
| 1인 개발 | **Android 100% 완성 (31세션)**, iOS 포팅만 필요 |

#### 현재 완성된 기능 (Android)
- **Health Connect 연동**: 삼성헬스/Google Fit → 심박수/걸음수/수면/혈압/산소포화도
- **실시간 생체 모니터링**: 30초 간격 + 배터리 최적화 자동 조절
- **위험 감지 → 긴급 SMS**: 연속 3회 위험 수치 → 가족에게 자동 GPS SMS
- **AI 건강 코칭**: Gemini 3 Flash (Firebase Cloud Functions) — 맞춤형 분석/리포트
- **건강검진 기록**: 국가검진 결과 연도별 비교 차트
- **가정 측정 기록**: 혈압/혈당/체중 트렌드 그래프 + 3일 연속 이상치 경고
- **AES-256 암호화**: SQLCipher + Android Keystore (100% 로컬 저장)
- **앱 잠금**: PIN/생체인증
- **6단계 온보딩**: 약관 → 인증 → 기본정보 → 질병 → 유전요인 → 목표

#### 출시를 위한 리프레이밍 (식약처/의료기기법 대응)

| 현재 표현 | 출시용 변경 |
|----------|-----------|
| "위험 감지 알고리즘" | "사용자 설정 범위 이탈 알림" |
| "서맥/빈맥/고혈압 위기" | "설정 범위보다 낮음/높음/크게 초과" |
| "AI 건강 분석" | "AI 생활 습관 리포트" |
| "의료 가이드라인 기반" | "일반 건강 정보 참고" |

#### 스모킹건 기능

**"AI 맞춤 코칭 + 긴급 SMS + 암호화 로컬 저장"** — 삼성헬스에 없는 3종 세트

| 기능 | 삼성헬스 | WayPulse |
|------|---------|----------|
| AI 맞춤 건강 분석/조언 | **없음** | Gemini AI 코칭 |
| 위험 시 가족 SMS | **없음** | 자동 GPS SMS |
| 건강검진 기록 비교 | **없음** | 연도별 차트 |
| 데이터 저장 | 삼성 서버 전송 | **100% 로컬 + AES-256** |
| 가격 | 무료 (데이터 대가) | ₩6,000/월 (프라이버시) |

| 기능 | Cardiogram ($99/yr) | Welltory ($80-156/yr) | WayPulse (₩60,000/yr ≈ $44) |
|------|---------------------|----------------------|---------------------------|
| AI 코칭 | 없음 | 제한적 | **Gemini 풀 코칭** |
| 긴급 SMS | 없음 | 없음 | **자동 가족 SMS** |
| 검진 기록 | 없음 | 없음 | **국가검진 비교** |
| 가격 | $99/yr | $80-156/yr | **$44/yr (50-70% 저렴)** |

#### 핵심 연령층 & 충성 고객

| 그룹 | 연령 | 특징 | 구독 의향 |
|------|------|------|----------|
| **만성질환 관리자** | 45-70세 | 고혈압/당뇨 매일 측정, 기록 필수 | ★★★★★ (생존) |
| **스마트워치 헬스족** | 30-50세 | Galaxy Watch/Pixel Watch 착용, 데이터 활용 욕구 | ★★★★☆ (가치) |
| **자녀→부모 선물** | 30-45세 자녀 | "아버지 혼자 사시는데 위험하면 SMS 오게" | ★★★★★ (안심) |
| **프라이버시 중시** | 전 연령 | "내 건강 데이터를 삼성/구글에 주기 싫다" | ★★★★☆ (신념) |

> **핵심 수익층: 45-65세 만성질환자 + 스마트워치 착용자**
> **이탈률 최저 그룹: 자녀→부모 선물 구독 (부모님 건강 = 해지 불가)**

#### 왜 WayBack DNA인가
- WayBack: "산에서 길을 잃으면" → 안전 복귀 → **자연의 위험**
- WayPulse: "몸에서 이상 신호가 오면" → 안전 알림 → **몸의 위험**
- **둘 다 "위험할 때 곁에 있는 앱"**. 있으면 좋은 게 아니라 **없으면 죽을 수 있는** 앱.

### 4. WayScan (웨이스캔) — 식품 알레르기 스캐너

> **"아이가 땅콩 알레르기인데, 이 과자에 뭐가 들었는지 모르겠다. 외국어라 읽을 수도 없다."**

**법적**: "식품 성분 정보 제공 도구" — 의료 아님

| | |
|---|---|
| DNA 순간 | 알레르기 아이 간식 고를 때, 해외 식당에서 주문할 때 |
| 없으면? | **아나필락시스 → 사망 가능**. 생사 문제 |
| 핵심 연령층 | **25-45세 알레르기 자녀 부모** |
| 벤치마크 | **Fig: $40/yr**, Yuka: $10-20/yr |
| 대기업 위협 | 없음 (Google/Apple은 알레르기 스캐너 안 만듦) |
| 스모킹건 | **카메라로 성분표 → AI가 14개 언어로 읽고 알레르기 경고** |
| 구독 전환 포인트 | "아이 생명이 걸렸는데 ₩3,000/월? 당연히 낸다" |
| 충성 고객 확충 | 알레르기 부모 카페, 소아과 대기실, 해외 여행 커뮤니티 |
| 1인 개발 | 6주 MVP (카메라 OCR + Gemini + 알레르기 DB) |
| 출시 | **7개월** |

#### 핵심 기능
- 식품 포장 사진 → AI가 14개 언어로 성분표 읽고 알레르기 성분 경고
- 바코드 스캔 → Open Food Facts DB
- 내 알레르기 프로필 등록 (땅콩, 갑각류, 유제품, 글루텐 등)
- 위험 성분 발견 시 빨간 경고 + 진동
- 식당 메뉴 사진 → AI 알레르기 추정
- **오프라인 모드** (해외 여행)
- 가족 프로필 공유 (아이 알레르기 → 할머니 앱에도 동기화)

### 5. WayPanic (웨이패닉) — 위급 상황 자동 대응

> **"밤길에 누가 따라온다. 증거를 남기고 가족에게 알려야 한다."**

**법적**: "개인 안전 유틸리티"

| | |
|---|---|
| DNA 순간 | 야간 귀가, 스토킹, 택시에서 불안, 혼자 여행 |
| 없으면? | 증거 없음, 도움 요청 못 함, 위치 알 수 없음 |

| 벤치마크 | **Noonlight: $5-10/월** (시장점유율 18%), bSafe: $2/월 |

| 대기업 위협 | Apple SOS 있지만 **자동 녹화+증거 보존은 없음** |
| 스모킹건 | **원터치 패닉 = 영상녹화+GPS전송+소리녹음+문자발송 동시** |

| 1인 개발 | 8주 MVP (WayBack SOS 재활용 + 카메라 + 녹음) |

#### 핵심 기능
- 원터치 패닉 모드: 자동 영상녹화 + GPS 실시간 전송 + 주변소리 녹음 + 문자 발송
- 은밀 모드: 화면 꺼진 상태에서도 녹화+위치 전송
- 가짜 전화 기능: 위장 통화로 위험 모면
- 귀가 안전 타이머: "30분 안에 도착 안 하면 자동 알림"
- 오프라인 SOS (WayBack 기술)

#### Noonlight/bSafe 대비 차별화
| 경쟁앱 | WayPanic |
|---|---|
| 영어/미국 중심 | **14개 언어, 아시아 시장** |

## 스모킹건 분석 — 전체 포트폴리오

### 최대 기회: WayPulse + WaySound (쌍두마차)

**WayPulse가 WaySound와 함께 최대 수익원인 이유:**

1. **이미 완성됨** — 다른 앱은 0에서 시작, WayPulse는 **Android 출시만 하면 됨**
2. **건강 앱 = 가장 높은 구독 전환** — 건강앱 다운로드→구독 전환율 **9.4%** (전 카테고리 1위)
3. **건강 앱 = 가장 높은 수익** — Revenue Per Install $0.44 (전 카테고리 1위)
4. **만성질환자 = 이탈률 최저** — 매일 혈압 기록 → 365일 사용 (시즌성 없음)
5. **자녀→부모 선물** — "아버지 혼자 사시는데 위험하면 알림 오게" = 해지 불가
6. **mHealth 시장 폭발 성장** — $37.5B (2024) → $86.4B (2030), **CAGR 14.8%**
7. **한국 디지털 헬스 시장** — **$24.6B** (2024), 스마트워치 보급률 급증

> **WayPulse 1개 앱이 나머지 10개 합친 것보다 클 수 있다**
> (건강 = 가장 돈을 많이 쓰는 카테고리, 이미 완성, 매일 사용)

## 브랜드 통합 메시지

**Safe Way — 당신이 위험할 때, 곁에 있는 앱**

```
길을 잃었을 때 — WayBack이 데려다줍니다
몸이 위험할 때 — WayPulse가 알려줍니다
말이 안 들릴 때 — WaySound가 보여줍니다
음식이 위험할 때 — WayScan이 알려줍니다
누가 따라올 때 — WayPanic이 지켜줍니다
기억이 흐려질 때 — WayBrain이 깨워줍니다
잠을 못 잘 때 — WayRest가 들어줍니다
야생이 위험할 때 — WayWild가 알려줍니다
해외에서 아플 때 — WayER이 도와줍니다
이야기가 사라질 때 — WayStory가 남깁니다
사고가 났을 때 — WayCrash가 안내합니다
```

> **모든 앱이 하나의 DNA를 공유: "위험하거나 곤란한 순간, 이 앱이 없으면 안 된다"**
